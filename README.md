# 😂 Jokes Web App

A full-stack web application built using **ASP.NET Core MVC** that allows users to create, manage, and explore jokes in a simple and interactive way.

---

## 🧾 Description

The Jokes Web App is designed to demonstrate core web development concepts including CRUD operations, database integration, and user authentication.

Users can add new jokes, edit existing ones, and delete them بسهولة من خلال واجهة استخدام بسيطة وسلسة. التطبيق يعتمد على قاعدة بيانات لتخزين البيانات بشكل منظم، مما يعكس سيناريو حقيقي لتطبيقات الويب.

---

## 🚀 Features

* ➕ Add new jokes
* ✏️ Edit existing jokes
* ❌ Delete jokes
* 📋 View all jokes
* 🔐 Authentication & Authorization
* 💾 Database integration

---

## 🛠️ Technologies Used

* **C#**

## 📂 Project Structure

* **Models** → Data structure & database entities
* **Views** → UI pages
* **Controllers** → Application logic
* **Data** → Database context

---

## ⚙️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/RamezSameh/JokesWebApp.git
```

2. Open the project in **Visual Studio**

3. Update the database:

```
Update-Database
```

4. Run the project
